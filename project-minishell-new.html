Minishell

Shell Unix complet - Projet personnel en équipe de 2

Détails du projet

Projet Personnel

Objectif du Projet

Recréer un shell Unix complet en C pur, capable de parser et exécuter des commandes avec support complet des pipes, redirections, gestion de signaux et environnement. L'objectif était de maîtriser les concepts bas niveau : fork/exec, gestion des descripteurs de fichier, et synchronisation de processus.

Projet réalisé en équipe de 2 personnes avec communication très fréquente pour éviter les conflits et sabotages mutuels.

Défis Rencontrés et Solutions

Défi : Coordination en Équipe de 2

Travail en parallèle sur même codebase. Risque énorme de conflits : deux personnes modifiant même fonction, merges difficiles, régressions. Communication non-optimale = projet qui s'effondre.

Solution : Daily standups, pair programming pour parties critiques, branches Git bien isolées, revue de code rigoureuse avant merge, et tests systématiques.

Défi : Parsing Robuste

Gérer quotes, échappements, espaces, délimiteurs. Parser naïf échoue sur cas réels : "echo 'hello world'" vs echo hello world. State machine complexe.

Solution : Automate d'état pour tracker quotes, séparation tokenisation/parsing, tests exhaustifs sur cas limites.

Défi : Gestion des Signaux

Ctrl+C doit tuer enfant, pas shell. Ctrl+\ génère core dump. Signaux asynchrones, race conditions possibles.

Solution : Handlers SIGINT/SIGQUIT qui affectent seulement foreground process group, SIGCHLD pour reaper enfants.

Défi : Pipes Multiples

cmd1 | cmd2 | cmd3 requiert gestion précise des descripteurs. Fork pour chaque commande, rediriger les pipes. Risque de deadlock ou fuite de descripteurs.

Solution : Boucle systématique fork/exec pour chaque commande, close imédiat des descriptors inutiles, waitpid pour tous les enfants.

Extrait de Code

int	main(int ac, char **av, char **env)
{
	char				*line;
	t_list				*tokens;
	t_ast				*ast;
	t_list				*env_list;
	static int			last_exit_status;

	env_list = init_env_list(env);
	last_exit_status = 0;
	init_signals();
	if (ac == 2)
		test(av[1], &env_list, last_exit_status, 1);
	else if (ac >= 3 && !ft_strncmp(av[1], "-c", 3))
	{
		int exit_status = test(av[2], &env_list, last_exit_status, 0);
		exit(exit_status);
	}
	while (1)
	{
		line = read_line();
		tokens = token(line, env_list, last_exit_status);
		ast = parse(tokens);
		last_exit_status = execute(ast, &env_list, last_exit_status);
		free(line);
		ft_lstclear(&tokens, free_token);
		ast_free(ast);
	}
	ft_lstclear(&env_list, free_env_var);
	return (EXIT_SUCCESS);
}
