Serveur HTTP multiclient en C++98 - Projet personnel en équipe de 2

Détails du projet

Projet Personnel

Objectif du Projet

Implémentation d'un serveur HTTP complet conforme à HTTP/1.1. Capable de gérer plusieurs clients simultanément sans bloquer grâce à l'utilisation de poll(). Parser robuste des requêtes, gestion des routes, support CGI et génération de réponses HTTP complètes.

Projet réalisé en équipe de 2 personnes avec forte coordination requise pour avancer efficacement en parallèle.

Défis Rencontrés et Solutions

Défi : Coordination en Équipe pour Projet Complexe

Projet volumineux avec nombreuses dépendances. Deux personnes doivent travailler sans se bloquer mutuellement. Interfaces doivent être claires, sinon intégration devient cauchemar.

Solution : Design d'interfaces strictes dès le départ, modularisation (Server, Socket, Request, Response, etc.), tests unitaires pour chaque module, intégration progressive.

Défi : Non-bloquant avec poll()

poll() retourne événements mais gestion devient complexe : state machines par client, lecture partielle, écriture partielle. Sinon deadlock ou client bloqué.

Solution : State machine par client (READING, PARSING, SENDING, CLOSED), buffers circulaires avec offsets, continue opération au poll() suivant.

Défi : Parser HTTP Robuste

Headers multiples, chunked encoding, multipart form-data, bodies vides ou partiels. Parser naïf échoue vite.

Solution : Parser incrémental qui traite headers puis body séparément, validation stricte, gestion des cas limites.

Défi : Gestion CGI Non-bloquant

Exécuter scripts CGI sans bloquer serveur. Fork coûteux, gestion stdout complexe, timeouts nécessaires.

Solution : Fork au besoin, rediriger stdout vers pipe, ajouter à poll(), lire progressivement, timeout si délai dépassé.

Extrait de Code

void Server::run()
{
    std::vector<pollfd> fds;
    
    while (g_running)
    {
        fds.clear();
        for (size_t i = 0; i < _clients.size(); ++i)
            fds.push_back(_clients[i].getPollFd());
        
        int poll_ret = poll(fds.data(), fds.size(), TIMEOUT_MS);
        if (poll_ret < 0)
            throw std::runtime_error("poll error");
        
        for (size_t i = 0; i < fds.size(); ++i)
        {
            if (fds[i].revents & POLLIN)
                _clients[i].handleRead();
            if (fds[i].revents & POLLOUT)
                _clients[i].handleWrite();
            if (fds[i].revents & (POLLERR | POLLHUP))
                _clients[i].close();
        }
    }
}
